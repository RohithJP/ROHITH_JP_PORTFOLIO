Rsyslog EC2 CloudWatch Linux Monitoring

Project Overview

Built a centralized log management system where multiple EC2 servers forward logs to a dedicated log collector via Rsyslog. The collector categorizes logs using hostname-based templates, and CloudWatch ingests these logs for monitoring, alerting, and audit visibility.

What I Troubleshooted

  • Misconfigured Rsyslog modules causing no forwarding.
  • Port 514 blocked in security groups.
  • CloudWatch agent not detecting remote log directories.
  • Hostname mismatch creating duplicate folders.
