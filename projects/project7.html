07 — Two-Tier Architecture (Web + Private DB)

Secure VPC design: public EC2 web server talks to private EC2 MySQL database. Emphasis: SG/NACL design, MySQL host rules, troubleshooting runbook.

Architecture

Internet → Public Web EC2 → Private DB EC2 (MySQL)

• Public EC2: Nginx + PHP
• Private EC2: MySQL only
• SG reference used to allow only web→db traffic
• NACLs control subnets (stateless)
