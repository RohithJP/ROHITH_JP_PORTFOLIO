L1 — Linux System Administration & Fundamentals

Hands-on Linux project covering 25+ real-world system administration tasks — users, groups, permissions, services, cron, networking, file systems, logs, and troubleshooting.

Core Areas

Project Overview

This Linux project demonstrates complete system administration capabilities across users, groups, access control, systemctl services, logs, cron jobs, permissions, file systems, archiving, text processing, and process monitoring. It includes real-world troubleshooting scenarios using core command-line tools.

Goal

Build a strong foundation in Linux by executing real sysadmin tasks that mirror production operations: account management, security permissions, service troubleshooting, automation, and system monitoring.

Major System Administration Tasks

  • Create, modify, lock, and delete users and groups
  • Manage sudo privileges and secure sudoers configurations
  • Adjust file ownership and permissions using chmod/chown
  • Monitor processes using ps, top, htop, systemctl
  • Start, stop, enable and debug services
  • Identify heavy CPU/RAM processes
  • Analyze logs in /var/log and filter with grep
  • Schedule jobs with cron and verify logs
  • Handle file compression (tar, gzip)
  • Use grep, sed, awk for text filtering and automation

Key Troubleshooting Scenarios

  • Fixing broken services using journalctl + systemctl
  • Resolving permission-denied issues
  • Fixing PATH and environment variable errors
  • Investigating background processes & killing stuck ones
  • Diagnosing cron jobs that fail silently

Techniques Learned

  • Using systemctl and journalctl effectively
  • Process control and job management
  • Regex and CLI text processing
  • Secure user administration
