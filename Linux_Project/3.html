Linux Security & Password Extraction (CTF-Style)

Solved multi-level encoding and credential puzzles using Linux tooling and forensic techniques.

Project Overview

A collection of structured security challenges inspired by OverTheWire Bandit and real-world forensic tasks: decoding, searching hidden flags, and analyzing file permissions.

Key Activities

  • Decoded Base64, hex, and reversed text payloads using CLI tools.
  • Used find, strings, and xxd to locate hidden credentials.
  • Analyzed file permissions and ownership to infer escalation paths.
  • Documented reproducible steps and commands for each challenge level.

Technologies & Tools

Linux CLI grep / find xxd / strings Base64 / Encoding SSH-based challenges
