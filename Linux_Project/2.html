Ubuntu Server Hosting & Web Application Deployment

End-to-end deployment and troubleshooting of web apps on Ubuntu — Node.js, NGINX, firewall, ports and storage issues.

Project Overview

Deployed production-style environments on Ubuntu and resolved real hosting issues including service failures, package lock conflicts and network debugging.

Responsibilities & Tasks

  • Deploy Node.js app behind NGINX reverse proxy and configure systemd service files.
  • Diagnose "waiting for cache lock" apt errors and safely recover package manager state.
  • Resolve webpage not loading — analyze ss -tulnp, logs, and port bindings.
  • Fix storage-related failures and adjust server disk usage & cleanups.
  • Setup basic firewall rules and test endpoint connectivity using curl.

Outcome

Stable and repeatable deployment playbook with documented fixes and monitoring commands for production readiness.

Technologies
Ubuntu NGINX Node.js systemd
Deliverables
  • Deployment guide (playbook)
  • Recovery steps for apt & cache lock issues
  • Port and service diagnostic cheatsheet
